# AI Prompt Spec Lab

## Goal
Learn how to turn vague AI requests into structured, precise specifications that improve output quality and consistency.

## Why This Matters
Most AI failures come from unclear instructions. This project focuses on improving clarity, structure, and reliability.

## Sections
- before-after: vague vs structured prompts
- templates: reusable prompt frameworks
- comparisons: results across different AI models

## First Exercise
Rewriting vague prompts into structured specifications.
